## Getting Started

FlutterFlow projects are built to run on the Flutter _stable_ release.

## Manage dependencies and compilations

### Upgrade Dart SDK

```sh
flutter upgrade
```

### Install dependencies

```sh
flutter pub get
```

### Build

```sh
flutter build appbundle --build-name <BUILD_NAME> --build-number <BUILD_NUMBER>
```
